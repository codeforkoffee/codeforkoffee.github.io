## Features

* Code Platoon
* bravo
* Disqus integration
* Syntax highlighting
* Optional post image
* Social icons
* Page for sharing projects
* Optional background image
* Simple navigation menu
* MathJax support

See a [live version of Moon](http://taylantatli.github.io/Moon) hosted on GitHub.
